[+] Страница signup/register - регистрация. Есть 3 поля: email, password, repeat password.
    Ожидается, что, только введя правильные данные, мы сможем создать пользователя, иначе, покажем ошибки.

[+] Login - страница входа в систему. 2 поля: email, password.
    Только если на “сервере” есть такой созданный пользователь, мы сможем войти в систему, то есть перейти на страницу 3.
    Далее, в авторизованной части приложения мы можем нажать кнопку logout, которая позволяет выйти из нашего приложения и попасть на страницу Login 2.

[+] Страница About - информация о пользователе. Включает дату регистрации пользователя.

[ ] Страница заметок пользователя, в которой отображаются все заметки, сделанные пользователем.
    Порядок отображения: сначала более новые. По нажатию на саму заметку, мы переходим на страницу просмотра заметки 7.
    Есть кнопка “создать новую заметку, которая направляет нас на страницу создания заметки 5.
    А также у каждой заметки есть кнопка Редактировать (иконка), отправляет на страницу 6 и кнопка Удалить (иконка), которая предлагает нам удалить заметку пользователя.

[ ] Страница создания заметки. Мы можем ввести название заметки и тело заметки.
    И нажать “сохранить”, чтобы создать заметку. Сохранение перенаправляет нас на страницу просмотра созданной заметки 7.
    Кнопка back возвращает нас на страницу всех заметок 4.

[ ] Страница редактирования заметки.
    Пользователь может изменить имя или/и тело заметки и нажать “Сохранить”, чтобы применить изменение к заметке.
    Сохранение перенаправляет нас на страницу просмотра созданной заметки 7. Кнопка back возвращает нас на страницу всех заметок 4.

[ ] Страница просмотра заметки.
    Есть кнопка Редактировать (иконка), отправляет на страницу 6 и кнопка Удалить (иконка), которая предлагает нам удалить заметку пользователя.
    Удаление перенаправляет нас на страницу заметок 4. Кнопка back возвращает нас на страницу всех заметок 4.

[ ] Страница 404, которая предлагает нам пройти в about (Home), то есть на страницу 3.
